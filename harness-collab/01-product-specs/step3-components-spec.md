# Step3 组件开发需求规格

> 版本: 1.0 | 日期: 2026-06-05 | 状态: 已确认

## 1. 模块概述

本阶段开发商品详情页的两个核心子组件：促销标签组件和倒计时组件，并在父组件中完成集成与父子传值验证。

## 2. 功能需求

### 2.1 促销标签子组件（PromoLabel.vue）

- 接收父组件传入的促销类型（如"限时秒杀"、"满减优惠"、"新品首发"、"热销爆款"），展示对应标签
- 不同促销类型对应不同颜色/样式（秒杀红色、满减橙色、新品蓝色、热销绿色）
- 若无促销信息则不显示标签

### 2.2 倒计时子组件（Countdown.vue）

- 接收父组件传入的结束时间（时间戳或日期字符串），计算剩余天/时/分/秒
- 每秒更新显示，到达结束时间后显示"活动已结束"
- 组件销毁时清除定时器防止内存泄漏

### 2.3 父组件集成（ProductDetail.vue）

- 展示商品基本信息（名称、价格、图片占位、描述）
- 集成促销标签组件：传入促销类型 prop
- 集成倒计时组件：传入活动结束时间 prop
- 验证父子组件传值：动态修改促销类型/结束时间，子组件响应变化

## 3. 组件树

```
ProductDetail (父组件)
├── PromoLabel (子组件 — 促销标签)
└── Countdown (子组件 — 倒计时)
```

## 4. 非功能需求

- 使用 Vue 3 Composition API
- 倒计时定时器必须在组件卸载时清理
- 父组件通过 props 向子组件传值
- 子组件通过 emit 向父组件通知事件（如倒计时结束）
