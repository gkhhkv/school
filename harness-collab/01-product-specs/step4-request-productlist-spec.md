# Step4 异步请求封装与商品列表分页 — 需求规格

> 版本: 1.0 | 日期: 2026-06-05 | 状态: 已确认

## 1. 模块概述

本阶段封装 Axios 异步请求工具模块，实现商品列表分页查询与分类筛选功能，并配置 Vite 开发代理连接后端 API。

## 2. 功能需求

### 2.1 Axios 请求工具封装（request.js）

- 创建 Axios 实例，配置 baseURL 为 `/api`
- 设置请求超时为 10000ms
- 请求拦截器：添加 Content-Type JSON 头
- 响应拦截器：解包后端统一返回格式 `{ code, message, data }`
  - code === 200 时返回 data
  - 其他 code 时 reject 并携带错误信息

### 2.2 商品列表分页查询

- 调用 GET /api/products?pageNum=&pageSize=&keyword= 接口
- 展示商品卡片（图片、名称、价格）
- 底部分页条：上一页/下一页按钮 + 当前页/总页数显示

### 2.3 分类筛选

- 提供分类按钮：全部/数码/服装/食品/家居
- 点击分类按钮将对应关键词填入 keyword 参数并发起查询
- 支持与分页同时生效

## 3. 非功能需求

- Vite 开发代理转发 /api 请求到 Spring Boot 后端 (localhost:8080)
- 分页按钮在首/末页时正确禁用
- 请求中/无数据/请求失败三种状态有对应 UI 呈现
