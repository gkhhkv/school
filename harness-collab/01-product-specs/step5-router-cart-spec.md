# Step5 Vue Router + Pinia 购物车 — 需求规格

> 版本: 1.0 | 日期: 2026-06-05 | 状态: 已确认

## 1. 模块概述

本阶段为前端引入 Vue Router 实现多页面导航，引入 Pinia 实现购物车状态管理，打通"浏览商品 → 加入购物车 → 下单成功"的完整流程。

## 2. 功能需求

### 2.1 Vue Router 路由配置

- 首页 `/`：商品列表（ProductList）
- 购物车 `/cart`：购物车页面（ShoppingCart）
- 下单成功 `/order-success`：下单成功页面
- 404 页面：匹配所有未定义路由

### 2.2 Pinia 购物车 Store

- 添加商品到购物车（重复添加则数量+1）
- 从购物车移除商品
- 修改商品数量（最小1）
- 清空购物车
- 实时计算购物车总件数和总金额

### 2.3 购物车页面

- 展示已添加商品列表（名称、单价、数量、小计）
- 每项提供 + / - 按钮调整数量，以及删除按钮
- 底部显示总件数、总金额
- 清空购物车按钮
- 提交订单按钮 → 清空购物车 → 跳转下单成功页面

### 2.4 下单成功页面

- 显示"下单成功"提示
- "返回首页"链接

### 2.5 404 页面

- 显示"页面不存在"提示
- "返回首页"链接

## 3. 页面流转

```
首页(/) ──点击商品──→ 加入购物车(Store)
  │                      │
  ├──导航栏──→ 购物车(/cart) ──提交订单──→ 下单成功(/order-success)
  │                │                          │
  └──← 返回首页 ←──┴────────← 返回首页 ←──────┘
```
